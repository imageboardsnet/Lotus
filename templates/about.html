About ImageBoards.net

Thank you for visiting my website!

This is a side project I developed to help people in discovering imageboards. Initially, this project began as a frontend for the cdd0 imageboards.json project. However, since it was abandoned in 2022, I have been maintaining my own list. All the source code is available here: https://github.com/imageboardsnet
