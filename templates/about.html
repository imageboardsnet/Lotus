About ImageBoards.net

Thank you for visiting ImageBoards.net!

This is a project I created to help people discover imageboards. It originally started as a frontend for the ccd0 imageboards.json project. After that project was abandoned in 2022, I began maintaining my own list.

You can easily add your own imageboard or one that is not yet listed. To do this, visit https://blossom.imageboards.net/. You can register with one click and will receive a randomly generated login and password. After that, you’ll be able to freely add an imageboard to the list or claim an existing one. Please note that every new imageboard is manually verified; this usually takes place every 1–2 days.

Feel free to contact me at: me[@]imageboards.net

Donations

I develop and host this website for free, and the source code is released under the GPLv3 license. I don’t want to add ads, as they would make the site worse while generating almost no revenue. If you’d like to support the project, you can donate with cryptocurrency.
